# 1. 📊 Reusable Trend Analysis Framework

Framework ini digunakan untuk menganalisis teknologi baru secara sistematis.

## 🔍 1. Identifikasi Tren
- Apa nama teknologinya?
- Apa konsep utamanya?
- Termasuk kategori apa? (AI, IoT, Web, Mobile)
- Status tren: emerging / growing / mature

**Contoh:**
> On-device AI (AI yang berjalan langsung di HP tanpa internet)

---

## 📌 2. Bukti Adopsi Teknologi
Cek apakah tren ini benar-benar digunakan:

- Demo di event besar (Google I/O, Flutter Forward)
- Paper ilmiah (arXiv)
- Sudah dipakai di aplikasi nyata
- Ada library/framework resmi

---

## ⚖️ 3. Pendorong & Hambatan

### ✔️ Pendorong
- Perangkat semakin kuat
- Privasi lebih penting
- Proses lebih cepat (tanpa server)
- Dukungan framework modern (Flutter, AI toolkit)

### ❌ Hambatan
- Baterai cepat habis
- Ukuran model besar
- Performa terbatas di device low-end
- Kompleksitas implementasi

---

## 💡 4. Peluang untuk Flutter Developer

Dengan Flutter, peluangnya:

- Integrasi AI (TFLite / ONNX)
- Aplikasi offline AI
- Smart camera / object detection
- Personal assistant di mobile
- Aplikasi IoT dashboard

---

## 🧭 5. Rekomendasi

- Cocok untuk dipelajari jika ingin fokus mobile dev
- Prioritas tinggi untuk masa depan
- Skill penting:
  - Flutter + Dart
  - API AI / ML mobile
  - Optimasi performa

---

# 2. 🎤 Workflow Future Tech Talk

## 1. Pilih Topik
- Pilih tren yang relevan (AI, IoT, PWA, Super Apps)
- Pastikan ada data & contoh nyata

---

## 2. Riset Sumber
Gunakan sumber terpercaya:
- Google I/O
- Flutter Forward
- arXiv (paper ilmiah)

---

## 3. Buat Narasi
Struktur cerita:
- Problem → masalah saat ini
- Solution → teknologi baru
- Impact → dampak ke industri
- Opportunity → peluang developer

---

## 4. Buat Slide
- 1 slide = 1 ide
- Gunakan gambar & diagram
- Hindari teks panjang

---

## 5. Latihan Presentasi
- Latihan minimal 2–3 kali
- Fokus pada alur dan timing

---

## 6. Rekam Presentasi
Gunakan:
- Loom
- OBS Studio

Pastikan:
- suara jelas
- slide full screen

---

## 7. Upload & Share
- YouTube / Google Drive
- GitHub
- Forum kampus / LinkedIn
